#VERGE CURRENCY
バージカレンシー

#Quick Guide
クイックガイド　

# completely anonymous and privacy focused :
完全に匿名でプライバシーに特化
=> 完全な匿名性とプライバシーの実現にフォーカス

バージは、Torとi2pといった匿名中心の複数のネットワークを使用します。ユーザーのIPアドレスは読み取り困難となり、取引内容は完全に追跡不可能です。

=> バージは、TorやI2Pに代表される、ユーザーの匿名性の確保を根幹的な目標とするネットワークを利用し、
ユーザーのIPアドレスは難読化され、トランザクションの内容を追跡することも、完全に不可能です。

# great support from community :
コミュニティーによるすばらしいサポート
=> 強固なコミュニティによるサポート

バージは私企業ではなく、プレマインやICOによる資金集めもしていません。全ての開発やマーケティングは数千人のコミュニティーメンバーによって支えられています。

=> バージはプレマインやICOで調達した資金によって設立されたプライベートな企業ではありません。
開発やマーケティングなどの活動はすべて、数千人から構成されるコミュニティのメンバーによって支えられています。

# decentralized and open source :
非中央集権型でオープンソース
=>

バージはオープンソースプラットフォームにもとづいた非中央集権型通貨です。コインをコントロールする中心的存在はいません。
=> バージはオープンソースプラットフォームに支えられた、非中央集権な通貨です。バージのコインをコントロールする者は存在しません。

# active dev team with many contributors :
多くの協力者による活発な開発チーム
=> アクティブな開発チームと多くの協力者

開発チームは活発でいつもコミュニティーと繋がっています。もしバージについて支援したい場合や、何か新しいアイディアをお持ちなら、メインの開発者と簡単に連絡をとることができます。
=> 精力的に開発を続けるバージの開発チームは、常にコミュニティと密接な関係を保っています。仮にあなたがバージに何らかのサポートや新しいアイデアの提供を行いたい時は、
簡単に主要な開発メンバーと繋がることができます。

# ready for mass adoption :
幅広い受け入れ体制
=> より多くのユーザーへの普及可能性

バージは複数のオペレーティングシステムに対応した安全なウォレットを幅広く提供します。革新的な "Tor Android Wallet" は、モバイル端末での匿名性を提供しています。これはプライバシー特化型のコインの中でも特徴的な機能です。

=> バージは複数のオペレーションシステムに対応する安定したウォレットを幅広く提供します。
中でも、革新的な"Tor Android Wallet"はモバイル端末での匿名性を提供しています。これはプライバシー特化を提唱する暗号通貨の中でも非常に特徴的です。

# RSK smart contract :
RSKスマートコントラクト
=>

RSKがアクティベーションされたら、ユーザーはバージブロックチェーン上でスマートコントラクトを動作できるようになります。
=>現在準備中のRSKがバージのネットワーク上でアクティベートされれば、ユーザーはバージのブロックチェーン上でスマートコントラクトを実行することが可能になります。

# ultra fast transactions :
ウルトラ・ファスト・トランザクション
=>超高速な取引の実行（ウルトラ・ファスト・トランザクション）

他のコインと比較して、取引スピードが超高速です。
=>バージは他のコインと比較して、取引の実行スピートが超高速です。
